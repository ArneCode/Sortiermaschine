\doxysubsection{Ausführliche Beschreibung}
Ein Call der vom \mbox{\hyperlink{classCallHandler}{Call\+Handler}} aufgerufen werden kann. 

Wird nie selbst instanziert sondern nur Abgeleitete Klassen. Abgeleitete Klassen müssen eine run Funktion, die ausgeführt wird wenn der Call an der Reihe ist und eine is\+Done Funktion, die angibt ob der Call abgeschlossen ist implementieren
